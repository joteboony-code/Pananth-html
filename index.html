💡 วิธีใช้ / Tips

คู่มือ Flow การใช้งาน
สรุปขั้นตอนงานที่ใช้บ่อยของระบบห้องเช่า เปิดดูเมื่อลืมว่าต้องกดอะไรก่อน–หลัง
จำสั้น ๆ: เปิดรอบใหม่ → จดมิเตอร์ → ตรวจยอด → ส่งบิล → รับชำระ/ติดตาม → ปิดรอบ
🧭 Flow ประจำเดือนแบบย่อ
1) กด 🆕 เปิดรอบรับชำระใหม่ เพื่อเริ่มรอบใหม่ และปลดล็อกการจดมิเตอร์
2) กด 📝 จดมิเตอร์รอบบิลใหม่ หรือจดทีละห้องจากการ์ดห้อง
3) ตรวจยอดรวม/ยอดผิดปกติให้เรียบร้อย
4) ส่งบิลด้วย 🚀 ส่ง LINE โดยตรง รายห้อง หรือ 📣 ส่งยอดทั้งหมด
5) ผู้เช่าส่งสลิปทาง LINE หรือเจ้าของรับเงินผ่าน 💳 ยอดค้าง / รับชำระ
6) ใช้แจ้งเตือนค้างชำระ / นัดวันชำระตามจำเป็น
7) จบเดือนค่อยเข้า 🔐 ล็อก/ปิดรอบบิล และดูย้อนหลังใน 📚 ประวัติรอบบิล
📝 เมื่อจะจดมิเตอร์ ต้องทำยังไง
1) ถ้าปุ่มจดมิเตอร์ยังล็อกอยู่ ให้กด 🆕 เปิดรอบรับชำระใหม่ ก่อน
2) เลือกวิธีจด:
• จดหลายห้อง: กด 📝 จดมิเตอร์รอบบิลใหม่
• จดรายห้อง: กดปุ่ม 📝 จดมิเตอร์ ที่การ์ดห้องนั้น
3) กรอกเลขมิเตอร์ ครั้งหลัง ของไฟและน้ำ
4) แนบรูปมิเตอร์ไฟ/น้ำได้ ถ้าต้องการเก็บหลักฐาน
5) ดู Preview ยอด แล้วกด ✓ บันทึก
หลังบันทึก ระบบจะคำนวณค่าเช่าใหม่ และห้องนั้นจะกลับเป็นสถานะ รอชำระ
✏️ จดมิเตอร์ผิด แก้ยังไง
1) แก้ได้เฉพาะ มิเตอร์ของรอบปัจจุบัน
2) เปิดการ์ดห้อง แล้วกด ✏️ แก้ไขมิเตอร์ หรือแก้จากหน้าจดมิเตอร์รวม
3) ระบบจะคงเลข “ครั้งก่อน” เดิมไว้ แล้วให้แก้เฉพาะเลข “ครั้งหลัง”
4) กดยืนยันการแก้ไข แล้วบันทึก
5) ถ้าเคยส่งใบแจ้งค่าเช่าไปแล้ว ให้ ส่งบิลใหม่ หลังแก้ไข
ถ้าห้องนั้นมีการรับชำระแล้วหรือจ่ายบางส่วนแล้ว ระบบจะกันไม่ให้แก้มิเตอร์ทับตรง ๆ เพื่อไม่ให้ประวัติยอดเพี้ยน
กรณีมีเงินเข้าแล้ว แต่ต้องปรับยอด ให้จัดการต่อใน 💳 ยอดค้าง / รับชำระ แทน
📣 แจ้งบิล / ส่งยอดให้ผู้เช่า
ส่งรายห้อง
1) แตะการ์ดห้อง
2) ตรวจยอดในห้องนั้น
3) กด 🚀 ส่ง LINE โดยตรง
4) จะเปิดดู 🧾 ใบแจ้งค่าเช่า ก่อนส่งก็ได้
ส่งหลายห้องพร้อมกัน
1) กด 📣 ส่งยอดทั้งหมด
2) ตรวจรายการห้อง ยอดคงเหลือ ห้องที่ไม่มี User ID และห้องที่ถูกงดแจ้ง
3) เลือกห้องที่จะส่ง แล้วกดส่ง LINE
4) ยืนยันอีกครั้งก่อนระบบส่งจริง
ถ้าระบบขึ้นป้ายเตือน เช่น มิเตอร์ลดลง / มิเตอร์สูงผิดปกติ / รอตรวจสลิป ควรตรวจห้องนั้นก่อนส่งบิล
💳 รับเงินสด / โอนนอกระบบ / โอนแต่ตรวจสลิปไม่ได้
กรณีผู้เช่าส่งสลิปทาง LINE และตรวจผ่าน ระบบจะบันทึกให้เอง
ถ้าต้องบันทึกเอง ให้ทำตามนี้:
1) เข้า 💳 ยอดค้าง / รับชำระ
2) เลือกห้อง และกรอกยอดที่รับจริง
3) เลือกวิธีรับเงิน: เงินสด / โอนนอกระบบ / โอนแต่ตรวจสลิปไม่ได้
4) ถ้าต้องการให้ผู้เช่ารู้ว่าสถานะอัปเดตแล้ว ให้ติ๊ก แจ้งผู้เช่าทาง LINE หลังบันทึก
5) ใส่หมายเหตุ แล้วดู Preview การตัดยอด
6) กด ✅ บันทึกรับเงินสด / โอนนอกระบบ
ระบบจะตัด ยอดค้างเก่าก่อน แล้วค่อยตัดยอดเดือนปัจจุบัน ถ้ายอดไม่ตรง ระบบจะถามยืนยันเพิ่มเพื่อกันกดผิดห้อง
📅 ผู้เช่านัดวันชำระ / งดแจ้งเตือนห้องนั้น
กรณีผู้เช่าแจ้งว่าจะจ่ายวันที่กำหนด
1) แตะการ์ดห้อง
2) กด 📅 นัดวันชำระ
3) ใส่วันที่ เช่น 25 และหมายเหตุ
4) ระบบจะข้ามแจ้งเตือนอัตโนมัติก่อนวันนัด
กรณีอยากจัดการหลายห้อง
เข้า 🔕 งดแจ้งเตือน เพื่อเลือกงดถาวร / นัดวันชำระ / ล้างสถานะงดแจ้ง
🚪 ผู้เช่าแจ้งออก ต้องทำยังไง
1) แตะการ์ดห้องของผู้เช่าคนนั้น
2) กด 🚪 แจ้งออกจากห้อง
3) กรอก/ตรวจ ค่าทำความสะอาด, ค่าเสียหายเพิ่มเติม, และหมายเหตุ
4) ดูสรุปว่าต้อง คืนเงินผู้เช่า หรือ ผู้เช่าต้องจ่ายเพิ่ม
5) ถ้าห้องมี LINE User ID ให้กด 📣 ส่ง LINE OA แจ้งผู้เช่า ก่อน
6) จากนั้นกด ✓ ยืนยันออกจากห้อง และตั้งเป็นห้องว่าง
หลังยืนยัน ระบบจะปิดแฟ้มผู้เช่าปัจจุบัน ย้ายข้อมูลเก่าไปประวัติผู้เช่า ตั้งห้องเป็นว่าง และล้าง LINE User ID ของห้องเดิม
🧍 รับผู้เช่าใหม่เข้าห้อง
1) กด 🧍 รับผู้เช่าใหม่
2) เลือกห้องที่เป็นสถานะ ห้องว่าง
3) กรอกข้อมูลผู้เช่า เบอร์โทร วันเข้าอยู่ สัญญา เงินประกัน และข้อมูลที่ต้องการ
4) แนบเอกสารได้ทันที หรือข้ามไปเพิ่มทีหลังใน 🧾 ผู้เช่า/เอกสาร
5) ตรวจสรุป แล้วกด ✅ ยืนยันรับผู้เช่าใหม่
ถ้าห้องยังไม่ว่าง มียอดค้าง หรือแฟ้มผู้เช่าเดิมยังค้างอยู่ ระบบอาจกันไม่ให้รับผู้เช่าใหม่ก่อน เพื่อป้องกันข้อมูลซ้อนกัน
🛠️ ผู้เช่าแจ้งซ่อมจาก Tenant Portal
1) เมื่อผู้เช่าแจ้งซ่อมจาก Tenant Portal ระบบจะเก็บงานไว้ในหลังบ้าน
2) เจ้าของเข้า 🛠️ งานแจ้งซ่อม
3) เปิดดูรายละเอียด ห้อง หมวดงาน ข้อความ และรูปแนบ
4) เปลี่ยนสถานะได้เป็น รอรับเรื่อง / กำลังทำ / เสร็จแล้ว / ปิดงาน
5) ใส่ 📝 หมายเหตุ ไว้กับงานได้
📢 ส่งประกาศถึงผู้เช่า
1) กด 📢 ประกาศผู้เช่า
2) พิมพ์ข้อความประกาศ หรือเลือก Template ที่เตรียมไว้
3) เลือกกลุ่ม/ห้องที่จะส่ง
4) ตรวจข้อความก่อน แล้วกด 📢 ส่งประกาศที่เลือก
🔐 ปิดรอบ / สำรองข้อมูล / ดูย้อนหลัง
1) ก่อนจบรอบ ให้ตรวจยอดค้าง รายการรับชำระ และห้องที่ยังต้องติดตาม
2) ระบบมี Auto Backup ก่อนงานสำคัญอยู่แล้ว แต่จะกด 🛡️ Backup เองเพิ่มก่อนปิดรอบก็ได้
3) เข้า 🔐 ล็อก/ปิดรอบบิล เพื่อตรวจสรุปและปิดรอบ
4) หลังปิดรอบ ดูย้อนหลังได้ที่ 📚 ประวัติรอบบิล
5) ถ้าต้อง Restore Backup ให้ตรวจชื่อไฟล์และช่วงเวลาที่ต้องการให้แน่ใจก่อน เพราะข้อมูลในระบบจะถูกเขียนทับ
🧪 ห้อง 99 ใช้เมื่อไหร่
ใช้ห้อง 99 สำหรับทดลอง Flow ต่าง ๆ โดยไม่กระทบห้องจริง
เช่น ทดลองจดมิเตอร์ ส่ง LINE ทดสอบ หรือเช็กการแจ้งออก
ยอดของห้อง 99 ไม่ควรนำไปรวมกับรายได้จริง
⚠️ ข้อควรระวังที่จำเป็น
• เดือนที่หน้าแรกเปลี่ยนเองไม่ได้ เพื่อกันส่งบิลผิดรอบ — เปลี่ยนรอบด้วยปุ่ม เปิดรอบรับชำระใหม่ เท่านั้น
• ถ้าแก้มิเตอร์หลังส่งบิลแล้ว อย่าลืมส่งบิลใหม่
• ถ้าผู้เช่าแจ้งออก ควรส่งสรุป LINE ก่อนกดยืนยันตั้งห้องว่าง
• ก่อน Restore Backup ต้องแน่ใจ เพราะข้อมูลใน KV จะถูกเขียนทับ
• ถ้าส่ง LINE ไม่เข้า ให้เช็ก Worker URL, LINE Token/OWNER_ID และหน้า Logs/ตรวจสุขภาพระบบ

📦 สำรองข้อมูลจาก Cloudflare KV
ดาวน์โหลดข้อมูลทั้งหมด เช่น ห้อง ผู้เช่า สถานะ ประวัติ ค่าใช้จ่าย และ logs เป็นไฟล์ JSON
☁️ Backup ไป Cloudflare R2
สำรองข้อมูลปัจจุบันจาก KV ไปเก็บเป็นไฟล์ JSON ใน R2 แบบแมนนวล กดเองเมื่อพร้อม เช่น ก่อนเปิดรอบใหม่ หรือหลังแก้ข้อมูลสำคัญ
ต้องผูก R2 Bucket กับ Worker โดยใช้ Binding ชื่อ RENTAL_R2 ก่อน ปุ่มนี้ถึงจะใช้งานได้
🤖 Auto Backup / ล้างไฟล์เก่า
ระบบจะ Backup ไป R2 อัตโนมัติก่อนงานสำคัญ เช่น เปิดรอบใหม่, Restore, รับชำระเอง, ยกเลิกรายการชำระ และรับสลิปที่ตรวจผ่าน
Auto Backup จะอยู่ใน backups/auto/ และ Worker จะพยายามลบไฟล์เก่ากว่า 180 วัน ทุกครั้งที่มี Auto Backup ใหม่
☁️ Restore จาก Cloudflare R2
โหลดรายการไฟล์ Backup ที่เก็บใน R2 มาให้เลือก Restore ได้ ไม่ต้องพิมพ์ path เอง
♻️ Restore ข้อมูลกลับเข้า KV
เลือกไฟล์ JSON ที่เคย Backup ไว้ ระบบจะเขียนทับข้อมูลใน KV
