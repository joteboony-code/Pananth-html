💡 วิธีใช้ / Tips

คู่มือใช้งานเร็ว
รวมขั้นตอนสำคัญของโปรแกรมห้องเช่า ใช้ดูเวลาลืมขั้นตอน หรือก่อนปิดรอบเดือน
✅ เริ่มต้นใช้งานประจำเดือน
1) เลือกเดือนที่ต้องการจากช่องเดือนด้านบน
2) กด 📝 จดมิเตอร์รอบบิลใหม่
3) ใส่เลขมิเตอร์ไฟ/น้ำของแต่ละห้อง
4) ตรวจยอดรวมหน้าแรกก่อนส่งให้ผู้เช่า
🏠 แก้ไขข้อมูลรายห้อง
1) แตะการ์ดห้องที่ต้องการ
2) แก้ชื่อผู้เช่า เบอร์โทร หมายเหตุ หรือค่า WiFi
3) ตรวจช่องคำนวณยอดด้านล่าง
4) กด ✓ บันทึก เท่านั้น ข้อมูลจึงจะถูกเก็บ
💳 รับชำระ / ยอดค้าง
ใช้เมนู 💳 ยอดค้าง/รับชำระ สำหรับรับเงินสดหรือโอนนอกระบบ
ระบบควรตัดยอดค้างเก่าก่อนเสมอ เช่น ค้างเดือนเก่า → ตัดเดือนเก่าก่อนเดือนใหม่
ถ้าเป็นสลิปจาก LINE ระบบจะตรวจผ่าน EasySlip แล้วบันทึกประวัติให้อัตโนมัติ
📦 ล็อกเดือน / ปิดรอบเดือน
1) ตรวจยอดรอบนี้ รับแล้ว และค้างชำระให้ครบ
2) กด 🛡️ Backup ก่อนปิดรอบทุกครั้ง
3) ค่อยกดล็อกเดือนหรือปิดรอบเดือน
4) หลังปิดรอบแล้ว ให้ดู 📚 ประวัติรอบบิล เพื่อตรวจข้อมูลย้อนหลัง
🧪 ห้องทดสอบ 99
ห้อง 99 ใช้ทดสอบระบบเท่านั้น
ไม่ควรรวมยอดกับรายได้จริง
การส่ง LINE ของห้อง 99 ให้ส่งเข้าเจ้าของ เพื่อไม่รบกวนผู้เช่าจริง
ถ้าจะทดสอบ EasySlip ห้อง 99 ต้องให้ Worker รองรับเลขห้อง 99 ด้วย
⚠️ ข้อควรระวัง
อย่ากด ↩️ ย้อนกลับปิดรอบล่าสุด ถ้าไม่ได้ปิดรอบผิดจริง
ก่อน Restore Backup ต้องแน่ใจ เพราะข้อมูลใน KV จะถูกเขียนทับ
ถ้ากดส่ง LINE แล้วไม่เข้า ให้ตรวจ Worker URL, LINE Token, OWNER_ID และ Logs

⚙️ ตั้งค่าระบบ

🔒 โหมดปลอดภัย

🛡️ Backup / Restore

📦 สำรองข้อมูลจาก Cloudflare KV
ดาวน์โหลดข้อมูลทั้งหมด เช่น ห้อง ผู้เช่า สถานะ ประวัติ ค่าใช้จ่าย และ logs เป็นไฟล์ JSON
☁️ Backup ไป Cloudflare R2
สำรองข้อมูลปัจจุบันจาก KV ไปเก็บเป็นไฟล์ JSON ใน R2 แบบแมนนวล กดเองเมื่อพร้อม เช่น ก่อนเปิดรอบใหม่ หรือหลังแก้ข้อมูลสำคัญ
ต้องผูก R2 Bucket กับ Worker โดยใช้ Binding ชื่อ RENTAL_R2 ก่อน ปุ่มนี้ถึงจะใช้งานได้
🤖 Auto Backup / ล้างไฟล์เก่า
ระบบจะ Backup ไป R2 อัตโนมัติก่อนงานสำคัญ เช่น เปิดรอบใหม่, Restore, รับชำระเอง, ยกเลิกรายการชำระ และรับสลิปที่ตรวจผ่าน
Auto Backup จะอยู่ใน backups/auto/ และ Worker จะพยายามลบไฟล์เก่ากว่า 180 วัน ทุกครั้งที่มี Auto Backup ใหม่
☁️ Restore จาก Cloudflare R2
โหลดรายการไฟล์ Backup ที่เก็บใน R2 มาให้เลือก Restore ได้ ไม่ต้องพิมพ์ path เอง
♻️ Restore ข้อมูลกลับเข้า KV
เลือกไฟล์ JSON ที่เคย Backup ไว้ ระบบจะเขียนทับข้อมูลใน KV
